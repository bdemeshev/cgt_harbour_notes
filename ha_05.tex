We do care only about which chairs are occupied.

\begin{enumerate}[label=\alph*)]
    \item In how many ways chairs can be occupied? 
    \item In how many ways chairs can be occupied if there should be at least one empty chair between occupied chairs?
    \item In how many ways chairs can be occupied if there should be at least two empty chairs between occupied chairs?
\end{enumerate}

    \item A bridge hand (random $13$ cards out of $52$-card standard deck) is dealt. 

\begin{enumerate}[label=\alph*)]
\item What is the probability that the hand contains no spades? 
\item What is the probability that the hand contains exactly $5$ hearts? 
\item What is the probability that the hand contains at most two aces? 
\end{enumerate}

Note: you can leave small number of binomial coefficients in the answer! Answer like ${16 \choose 11}/{25 \choose 13}$ is ok.

\begin{enumerate}
